código en la carpeta de src/components
* en Pages : existe todos los archivos javascript con los formularios y
en el método OnSubmit existe toda la lógica del formulario 
* en Data están los archivos jSon que contiene el arreglo con los diferentes 
objetos provenientes de los archivos excel de la tarea 1 
* en Navbar: es donde se desglosa el menú de las opciones 
* App.css : contiene los estilos de la aplicación 
* App .js : es la logica de la aplicación el menú con las rutas para abrir las páginas 
En la carpeta de algoritmos se encuentran algunas de las funciones usadas para llenar las tablas de frecuencia y probabilidad 
Los Data contienen json con las tablas de probabilidad y frecuencia 



Nota : para correrlo se debe tener instalado el node.js y el paquete npm 
, en nodeModules contiene lo necesario para correrlo , para ejecutar el programa 
se dirige a la consola y escribe el comando :  npm start  y al dar enter empieza la ejecución
se debe dirigir al localhost indicado en la consola para ver la página y usarla.
